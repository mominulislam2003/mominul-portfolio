# Mominul Islam Premium Portfolio

A cinematic React + Vite portfolio website for full stack developer Mominul Islam.

## Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- GSAP
- React Icons

## Run Locally

```bash
npm install
npm run dev
```

Then open the local URL shown by Vite.

## Notes

This workspace blocked process spawning during creation, so dependencies could not be installed or built here. Source-level checks were completed for file presence, local imports, ASCII text, and major styling hooks.
